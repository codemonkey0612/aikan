# Default Avatars

This directory contains default avatar images for different user roles.

## Required Images

Please add the following image files to this directory:

1. **admin.png** - Default avatar for admin users
   - Description: Pink hiragana characters "かあいい" (kaaii) in a square with pink border

2. **corporate.png** - Default avatar for corporate officers
   - Description: Professional man in suit with blue background

3. **facility-manager.png** - Default avatar for facility managers
   - Description: Woman cleaning with orange shirt and grey apron

4. **nurse.png** - Default avatar for nurses
   - Description: Woman in white lab coat, professional medical setting

## Image Specifications

- Format: PNG (recommended) or JPG
- Size: Square format (recommended: 200x200px or larger)
- Aspect Ratio: 1:1 (square)
- File naming: Must match exactly as listed above (case-sensitive)

## Usage

The avatars are automatically loaded based on user role when no custom avatar is uploaded.
The system will fall back to user initials if the default avatar image cannot be loaded.
